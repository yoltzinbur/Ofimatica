Clasificación del software

El software se clasifica según su función y forma de distribución:

Controla el hardware y permite ejecutar aplicaciones. Incluye:

  • Sistemas operativos (Windows, Linux, macOS)
  • Controladores o drivers
  • Utilidades de mantenimiento y seguridad

Diseñado para tareas específicas:

  • Ofimática: Word, Excel, PowerPoint
  • Diseño gráfico: Photoshop, Illustrator
  • Educativo: Duolingo, Khan Academy
  • Entretenimiento: Videojuegos, reproductores multimedia
  • Productividad: Navegadores, gestores de correo, software de contabilidad

Herramientas que permiten crear software:

  • Lenguajes de programación: Python, Java, C++
  • IDEs y editores: Visual Studio, Eclipse
  • Frameworks y librerías
  • Sistemas de control de versiones: Git, SVN
