Licencias de software

Las licencias determinan cómo se puede usar, modificar y distribuir un software.

Código cerrado, pago obligatorio o licencia limitada. El usuario no puede modificar ni redistribuir. Ejemplos: Windows, Adobe Photoshop, Microsoft Office.

Código abierto, gratuito, permite modificar, estudiar y redistribuir. Ejemplos: Linux, LibreOffice, GIMP. Fomenta colaboración y educación tecnológica.

Mezcla software libre y propietario: algunas partes son abiertas y otras requieren pago o restricciones. Ejemplos: versiones freemium de aplicaciones, software de código abierto con módulos comerciales.

Protegen derechos de autor, aseguran cumplimiento legal, fomentan la innovación y permiten a empresas y usuarios saber cómo pueden usar el software.
