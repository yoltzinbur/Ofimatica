Sistema Operativo Windows

Fundamentos del uso de Windows para el trabajo diario y la gestión eficiente del sistema operativo.

Windows es un sistema operativo desarrollado por Microsoft. Un sistema operativo (SO) es el programa principal que permite que la computadora funcione, ya que administra el hardware (procesador, memoria, discos, dispositivos) y permite que los programas se ejecuten.

Funciones principales de Windows:

  • 📂 Gestión de archivos: crear, copiar, mover y eliminar carpetas y documentos.
  • 🖥️ Interfaz gráfica: facilita el uso del equipo con ventanas, íconos y menús.
  • ⚙️ Gestión de recursos: administra memoria, procesador y dispositivos conectados.
  • 👤 Gestión de usuarios: permite crear cuentas y protegerlas con contraseña.
  • 🔌 Compatibilidad de software: ejecuta programas como Word, Excel, navegadores, juegos, etc.
  • 🌐 Conectividad: se conecta a internet y redes locales.
  1. Escritorio: pantalla principal que aparece al iniciar sesión. Contiene íconos de acceso directo y puede personalizarse.
  2. Barra de tareas: normalmente en la parte inferior, muestra programas abiertos, reloj, Wi-Fi, volumen y botón de inicio.
  3. Menú inicio: se abre al presionar el botón de inicio o la tecla Windows; permite abrir programas, apagar el equipo, buscar archivos y acceder a configuraciones.
  4. Íconos: pequeños accesos a programas o carpetas que se abren con doble clic.
  5. Ventanas: cada programa se abre en una ventana que puedes mover, maximizar, minimizar o cerrar.

🔌 Encender el equipo: Presiona el botón de encendido en la computadora y espera a que cargue Windows.

⏻ Apagar el equipo:

  1. Haz clic en el botón de Inicio.
  2. Selecciona Apagar o Suspender.
  3. Si no aparece, selecciona Icono de energía → Apagar.

📂 Abrir programas: Desde el menú inicio o escribiendo el nombre en la barra de búsqueda. Si está en el escritorio, doble clic sobre el ícono.

🖱️ Mover una ventana: clic y arrastra desde la barra superior.

⬜ Maximizar: clic en el ícono de cuadrado en la esquina superior derecha.

➖ Minimizar: clic en el ícono de guion (—) para ocultarla en la barra de tareas.

❌ Cerrar: clic en la X de la esquina superior derecha.

🎨 Cambiar fondo de pantalla:

  1. Clic derecho en el escritorio.
  2. Selecciona Personalizar.
  3. Entra en Fondo y elige imagen o color.

⏱️ Ajustar hora/fecha:

  1. Clic derecho sobre la hora en la barra de tareas.
  2. Selecciona Ajustar fecha y hora.
  3. Activa/desactiva hora automática o ajusta manualmente.

🌐 Cambiar idioma:

  1. Configuración → Hora e idioma.
  2. Selecciona Idioma y región.
  3. Agrega o cambia el idioma preferido.

Manejo de Archivos y Carpetas

Organización y gestión eficiente de documentos y carpetas en el sistema operativo

El Explorador de archivos es la herramienta principal para visualizar, abrir y gestionar archivos y carpetas.

  • Panel de navegación: Muestra accesos rápidos a Documentos, Imágenes, Descargas y Unidades de disco.
  • Barra de direcciones: Indica la ubicación actual de la carpeta y permite navegar escribiendo la ruta.
  • Panel principal: Donde se visualizan los archivos y carpetas contenidos en la ubicación actual.
  • Barra de búsqueda: Permite localizar archivos por nombre, tipo o fecha de modificación.

Las carpetas son contenedores donde se almacenan archivos. Puedes crear jerarquías de carpetas para mantener el orden:

  • Crear carpeta: Clic derecho → Nuevo → Carpeta. Asigna un nombre descriptivo.
  • Renombrar: Selecciona la carpeta → tecla F2 → escribe el nuevo nombre.
  • Buenas prácticas: Usa nombres claros, por ejemplo 2025_ProyectoVentas en lugar de “Nueva carpeta”.
  • Subcarpetas: Divide por categorías (por ejemplo: Documentos, Imágenes, Reportes).

Las operaciones más comunes para trabajar con archivos y carpetas son:

  • Copiar: Duplicar un archivo en otra ubicación (Ctrl + C).
  • Cortar: Mover un archivo a otro lugar (Ctrl + X).
  • Pegar: Colocar el archivo en la carpeta destino (Ctrl + V).
  • Eliminar: Selecciona el archivo y presiona Supr. Quedará en la Papelera de Reciclaje.
  • Restaurar: Desde la Papelera de Reciclaje puedes devolverlo a su ubicación original.

También puedes usar el ratón (clic derecho → copiar/cortar/pegar) o arrastrar archivos de una carpeta a otra.

Cada archivo tiene una extensión que indica su tipo. Ejemplos:

  • .docx → Documentos de Word.
  • .xlsx → Hojas de cálculo de Excel.
  • .pptx → Presentaciones de PowerPoint.
  • .txt → Archivos de texto plano (Bloc de Notas).
  • .jpg / .png → Imágenes.
  • .pdf → Documentos en formato universal de lectura.

Para encontrar archivos rápidamente:

  • Usa la barra de búsqueda del explorador de archivos.
  • Filtra por tipo de archivo (por ejemplo, escribe *.pdf para ver solo PDFs).
  • Filtra por fecha de modificación si recuerdas cuándo lo editaste.
  • Windows también puede buscar por contenido si el archivo es de texto.

Bloc de Notas

Editor de texto básico perfecto para notas rápidas y edición de archivos de texto simple

El Bloc de notas (Notepad) es un editor de texto muy ligero incluido en Windows. Trabaja con texto plano, lo que significa que no añade formato (negritas, tamaños, etc.) al guardar. Es ideal para tomar notas rápidas, editar archivos de configuración, escribir pequeños scripts y crear archivos HTML sencillos.

Formas comunes de abrir el Bloc de notas y crear archivos:

  • Desde el Menú Inicio: Haz clic en Inicio y escribe Bloc de notas (o Notepad) y ábrelo.
  • Ejecutar: Presiona Win + R, escribe notepad y presiona Enter.
  • Desde el Explorador de archivos: Clic derecho en un espacio vacío → Nuevo → Documento de texto. Esto crea un archivo .txt que puedes editar con Notepad.
  • Abrir archivos existentes: Dentro de Notepad: Archivo → Abrir y usa el selector de tipo para mostrar Todos los archivos (*.*) si quieres abrir .html, .css, .bat, etc.

Notepad permite escribir y editar texto plano. Algunas funciones útiles incorporadas:

  • Ajuste de línea (Word Wrap): hace que las líneas largas se visualicen en varias líneas en pantalla (no inserta saltos de línea en el archivo).
  • Cambiar fuente: Formato → Fuente para seleccionar tipo de letra y tamaño (solo afecta la vista, no el archivo).
  • Buscar y reemplazar: localizar texto con Ctrl + F o reemplazar con Ctrl + H.
  • Guardar codificación: al guardar puedes elegir la codificación (p. ej. UTF-8, ANSI).

Cómo guardar correctamente y qué opciones debes tener en cuenta:

  • Guardar rápido: Ctrl + S guarda el archivo con su nombre y ubicación actuales.
  • Guardar como: Archivo → Guardar como. Aquí puedes cambiar el nombre, la extensión y la codificación.
  • Extensión correcta: Para crear una página web guarda con .html (por ejemplo mipagina.html); para scripts guarda con .bat o la extensión correspondiente.
  • Tipo de archivo: En "Guardar como" selecciona Todos los archivos (*.*) si quieres escribir tu propia extensión (por ejemplo .html, .css, .js).
  • Codificación: Selecciona UTF-8 para máxima compatibilidad (recomendado para páginas web y archivos compartidos). Si tu sistema pide otra codificación, elige la que corresponda.

Ejemplo — crear una página HTML sencilla:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mi página de ejemplo</title>
</head>
<body>
  <h1>Hola desde Notepad</h1>
  <p>Este archivo fue creado y guardado desde el Bloc de notas.</p>
</body>
</html>

Guarda ese contenido como mipagina.html eligiendo "Todos los archivos" y UTF-8 en la codificación. Luego abre el archivo en un navegador para ver el resultado.

Ejemplo — crear un archivo por lotes (.bat):

@echo off
echo Hola mundo
pause

Guarda como hola.bat y ejecútalo con doble clic para ver la consola con el mensaje.

Notepad incluye herramientas básicas para localizar y cambiar texto:

  • Buscar: Ctrl + F abre la caja de búsqueda para localizar texto en el documento.
  • Reemplazar: Ctrl + H para buscar un texto y sustituirlo por otro (útil para cambios masivos).
  • Buscar siguiente/Anterior: puedes navegar por cada aparición del texto encontrado sin perder la posición actual.

Ejemplos de uso:

  • Tomar notas rápidas o listas de tareas.
  • Crear y editar archivos .txt, .html, .css, .js, .bat y otros archivos de texto simple.
  • Editar archivos de configuración (.ini, .conf) o revisar registros (logs).
  • Quitar formato de texto copiado desde otras aplicaciones.

Alternativas más potentes: si necesitas funciones avanzadas (numeración de líneas, búsqueda avanzada, pestañas, resaltado de sintaxis), considera usar Notepad++, Visual Studio Code o Sublime Text. Para documentos con formato enriquecido usa Word o WordPad.

Microsoft Office

Suite de productividad profesional líder en el mercado para crear documentos, presentaciones y hojas de cálculo

Word es el procesador de textos líder para crear documentos profesionales. Incluye herramientas de formato, revisión ortográfica, tablas, imágenes y plantillas para diversos tipos de documentos.

Excel es una potente hoja de cálculo para análisis de datos, creación de gráficos, tablas dinámicas y automatización mediante fórmulas y macros. Ideal para gestión financiera y análisis estadístico.

PowerPoint permite crear presentaciones profesionales con diapositivas, animaciones, transiciones y elementos multimedia. Incluye plantillas y herramientas de diseño para impactar en la audiencia.

Las aplicaciones de Office se integran perfectamente, permitiendo insertar hojas de Excel en Word, gráficos en PowerPoint y mantener vínculos dinámicos entre documentos.

LibreOffice

Suite ofimática gratuita y de código abierto, alternativa completa a Microsoft Office

Writer es el procesador de textos de LibreOffice, compatible con documentos de Word. Ofrece herramientas profesionales de formato, estilos, tablas de contenido y referencias cruzadas.

Calc es la hoja de cálculo que compite con Excel, ofreciendo funciones matemáticas, estadísticas, gráficos y tablas dinámicas. Compatible con archivos de Excel y otros formatos.

Impress permite crear presentaciones atractivas con efectos de transición, animaciones y plantillas prediseñadas. Compatible con archivos de PowerPoint para máxima flexibilidad.

LibreOffice es completamente gratuito, de código abierto, multiplataforma y ofrece actualizaciones regulares. Ideal para usuarios que buscan una alternativa económica a Microsoft Office.
